Garden Store App

Libraries and bookstores have electronic catalogs. So why not garden stores and nurseries too? You could search for the store's inventory of plants at computer terminals in the store. Or you could search your smartphone on the way to the store or while browsing in the store. This app is the beginnings of such a catalog.

  • Uses angularJS
  • Includes multiple search filters
  • Fully responsive
  • Includes a shopping cart for quick pre-purchase calculations
  • Features still being added

WeatherSmart Class Project

I developed this web page as a class project for the second in a series of courses in web development at Geekwise Academy. We were given a picture of what this webpage should look like. Then we had to re-create it on our own. I successfully created it so that it looks very much like the picture we were given. I also added gradients to make it look better. The web page is fully responsive. We were allowed and encouraged to use Bootstrap and Pure CSS. So I did. We also had to use Photoshop to extract an image of a woman from a background photo and touch the image up a bit. We also had to extract another smaller image from a background image. So this web page shows my ability to use Photoshop.
